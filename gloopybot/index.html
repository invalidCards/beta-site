GloopyBot Privacy Policy

GloopyBot is fully open-source. The code for it is hosted on Weird Gloop's Gitlab group and includes everything except the database of the public instance.

Under no circumstance are messages retained or transferred to me (invalidCards) or third parties in any way by the public instance of this bot. All messages are immediately processed by the bot purely to provide its functionality.

If a user uses the g!userwiki command to set a default wiki, the Discord snowflake of that user is stored in the database of the bot in order to save the set default and provide the functionality based on that default. It is not possible for users to add other users to the database in this way, nor are Discord snowflakes retained for users that do not have this feature enabled.

Discord snowflakes are used for each message that includes the linking syntax of the bot (text in [[square brackets]], {{curly brackets}} or --double dashes--) in order to look up whether the user has a user default set, but are in no way retained or transmitted if this is not the case.

If users wish to remove their Discord snowflake from the database, simply execute the g!userwiki default command in a channel where the bot can read messages, including direct messages. This removes the entire database entry with the user's Discord snowflake from the database.
