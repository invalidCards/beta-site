WikiaLinker will be shut down on November 7th, 2020. After that date, all user data will be deleted, and its GitHub repo will be archived.

Why?

On this date, Discord are modifying its backend to use their new domain (discord.com instead of discordapp.com). This change seems small, but it means that discord.js, the library behind the bot, has to be updated as well. There are many breaking changes in that upgrade process, and I honestly don't think it is worth my time anymore.

Now what?

If you're using the bot to link to the RuneScape Wiki or any of its sister wikis, consider using GloopyBot instead. It does all the same things, but for the new wiki.

You may also find use out of Wiki Operating Buddy, which links to a whole host of independently run gaming wikis.

Can I try to revive it?

Feel free! The bot is open source, and all the information is available to host it yourself. I just won't be doing it anymore.
