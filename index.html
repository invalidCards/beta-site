About

I am invalidCards, a programmer from the Netherlands. I mostly deal in JavaScript with NodeJS, as well as Microsoft Dynamics NAV and Microsoft Business Central, and can make games in the Unity and Godot engines.

Additionally, I can also find my way around Java, C#, HTML/CSS, PHP, SQL/MySQL/SQLite, and MongoDB, and have an affinity for topics related to quantum computing, security, malware and DevOps.

GloopyBot

GloopyBot is a bot for linking to the Weird Gloop family of wikis, such as the RuneScape and OldSchool RuneScape wikis, as well as to the English version of Wikipedia.

WikiaLinker

GooGlr

GooGlr was a somewhat simple bot that could quickly create goo.gl shortlinks to links supplied to it. This bot was shut down when the service did in March of 2019.

Birbbot

Birbbot is a bot commissioned by Aorpheat on Discord for the welcoming of new members, assigning them a role, and giving that role a random colour every time a new person joins their server.

Bugebot

Bugebot is a bot made for Kris on Discord. It assigns people a new name when prompted, chosen randomly from an extensive list of words from the English language.

Charbotte

Charbotte is a bot created for Charlotte on Discord. It reacts to messages with certain words in them, as well as randomly calling people non-sensical swears. (Note that this bot is entirely for comedic effect; it is not meant to insult or harass one single person in particular.)

Chungocraft

Chungocraft is a bot created for the Minecraft server of the same name. It tracks the amount of people online on the server and shows it as a custom status. Additionally, it also has a command for looking up the MOTD, player count and player list for any other Minecraft server.

Frob

Frob is a bot created for the Patreon Discord server of Filibusterfrog, allowing users in the server to change their name colour as well as certain notification roles at will, without requiring administrator intervention.

FroggoBot

FroggoBot is a bot created for Filibusterfrog's first Discord server, that included a few fun commands such as dice rolling and coin flipping. The bot is still online, but since the Discord server has seen very little activity in the past year, it is not expected that this bot will ever be worked on again.

Homeward Bot

HomewardBot is a clone of the Chungocraft bot created for the HomewardBound Minecraft server community. It shows the current playercount on the server in its custom status, and includes a command to look up the MOTD, playercount and player list for any Minecraft server.

Rosebot

Rosebot is a bot that I created for my own private Discord server. It adds some functionality that we were missing when we moved out of a DM group, such as notifications every time someone starts a voice call, and it also keeps a log of all deleted messages (in order to regulate the server's policy to allow anyone to manage any message) and automatically gives people their roles back after they've been kicked and have rejoined.

ShitPeopleBot

ShitPeopleBot is a bot created for the community of the Shit People Say in TF2 Tumblr blog. It allows people to set roles out of a list configurable by admins, as well as being able to scrape backpack.tf for in-game trading prices for Team Fortress 2 items.

TaskBot v2

TaskBot v2 is a bot created for the Taskman RuneScape community. It receives messages from the website whenever someone completes a task set by the system, creates a message to announce the completion in the appropriate Discord channel, and then sets the roles of the user according to their current progress through the tasks, provided that they have linked their Discord account on the website.

Terribot

Terribot was an experimental bot created for the RuneScape Wiki Discord server. It allowed anyone with a terrible idea (hence the name, Terrible Bot or Terribot for short) to suggest it using the bot, which would then make a GitHub issue out of it to save it, as well as allowing reactions to those ideas in the form of a GitHub reply. The bot is still online, but the channel created for this purpose has been deleted, and Terribot does not see a lot of use anymore; regardless, it was a good way for me to experiment with the limitations and capabilities of the GitHub API.

DistantQuest

DistantQuest is a fan-created follow-up of the game PesterQuest, which in turn is an extension to the web comic Homestuck. The game itself is classified as a "friend simulator", a type of visual dating simulator wherein the player tries to make friends with the characters. The project is written in RenPy and includes a lot of work from a lot of talented artists, musicians and other programmers, so be sure to check out the game and take a look at the credits in-game (some fandom knowledge required!).

FrogLeap

FrogLeap is a game prototype that I created during the My First Game Jam Summer 2018 edition. It is a platformer with procedurally generated levels. There was a plan to expand the game way further than it came in the end, with random effects applied to each level as well usable items, but this turned out to be too time-consuming for the time available during the game jam. In its current state it has a very basic game loop and is very unpolished and unfinished, but it is a concept that I would like to expand on in the future.

New XKit

New XKit is a Chrome and Firefox extension that functions as a toolkit for Tumblr, expanding on the default features of the site and introducing new ways to make the site more useable. The project has had over 70 contributors since its creatioin by a developer named Atesh, which the New XKit team in August of 2015 after he disappeared from the internet. At the time of writing, the project has approximately five active contributors, but anyone is welcome to add code to the project.
